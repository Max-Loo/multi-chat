# 国际化 (i18n) 功能测试计划

## 文档信息

- **创建日期**: 2026-03-12
- **版本**: 1.0
- **测试类型**: E2E (Playwright)
- **测试范围**: 国际化系统完整功能测试

## 1. 测试概述

### 1.1 测试目标

验证 Multi-Chat 应用的国际化系统功能完整性和用户体验一致性，包括：

- 语言切换功能的正确性
- 按需加载机制的可靠性
- 缓存验证和迁移逻辑
- Toast 队列的正确处理
- 自动持久化功能
- UI 文本多语言渲染
- 错误处理和降级策略
- 性能指标验证

### 1.2 系统架构

```
语言检测
    ↓
按需加载 (英文静态打包，其他异步加载)
    ↓
Toast 队列 (初始化期间提示)
    ↓
自动持久化 (Redux Middleware → localStorage)
    ↓
翻译完整性检查 (CI 集成)
```

### 1.3 支持的语言

| 语言代码 | 语言名称 | Flag |
|---------|---------|------|
| `zh`    | 中文    | 🇨🇳   |
| `en`    | English | 🇺🇸   |
| `fr`    | Français| 🇫🇷   |

### 1.4 核心命名空间

- `common` - 通用文本
- `chat` - 聊天功能
- `model` - 模型相关
- `navigation` - 导航菜单
- `provider` - 供应商
- `setting` - 设置页面
- `table` - 表格组件
- `error` - 错误消息（静态打包）

### 2.1 语言切换功能测试

#### 2.1.1 基本语言切换

**测试 ID**: `i18n-switch-001`

**测试目标**: 验证用户可以通过设置页面成功切换语言

**前置条件**:
- 应用已启动并完成初始化
- 用户处于设置页面

**测试步骤**:

1. 启动应用，等待初始化完成

**预期结果**:
- 语言切换 Toast 提示正确显示
- 所有 UI 文本正确更新为目标语言
- 下拉框显示当前选中的语言

- 设置页面标题显示正确语言

```typescript
// e2e/utils/i18n-helpers.ts

/**
 * 设置 localStorage 语言
 */
export async function setLocalStorageLanguage(page: Page, lang: string) {
  await page.evaluate((language) => {
    localStorage.setItem('multi-chat:language', language);
  }, lang);
}

/**
 * 获取当前 localStorage 语言
 */
export async function getLocalStorageLanguage(page: Page): Promise<string | null> {
  return page.evaluate(() => {
    return localStorage.getItem('multi-chat:language');
  });
}

/**
 * 清除语言相关 localStorage
 */
export async function clearLanguageStorage(page: Page) {
  await page.evaluate(() => {
    localStorage.removeItem('multi-chat:language');
  });
}

/**
 * 等待语言切换完成
 */
export async function waitForLanguageChange(page: Page, expectedLang: string) {
  await page.waitForFunction((lang) => {
    const stored = localStorage.getItem('multi-chat:language');
    return stored === lang;
  }, expectedLang);
}
```
