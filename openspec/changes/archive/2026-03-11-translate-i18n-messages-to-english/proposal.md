# 翻译 i18n.ts 中的硬编码消息为英文

## Why

`src/lib/i18n.ts` 作为国际化系统的核心模块，内部存在多处硬编码的中文字符串（Toast 消息和日志），这与国际化系统的职责不符。在 i18n 初始化阶段无法使用 `t()` 函数（鸡生蛋问题），因此需要将这些硬编码字符串统一改为英文，以保持一致性和避免国际化初始化依赖问题。

## What Changes

- **修改文件**: `src/lib/i18n.ts`
- 将所有硬编码的中文字符串翻译成英文：
  - Toast 消息（语言迁移提示、系统语言切换提示、错误提示等）
  - Console 日志（警告、错误信息）
- **非破坏性变更**：不影响外部 API 和功能行为

## Capabilities

### New Capabilities
无

### Modified Capabilities
无（此变更仅为实现细节优化，不涉及规范级行为变更）

## Impact

- **受影响代码**: `src/lib/i18n.ts` (约 7 处硬编码中文字符串)
- **用户体验**: Toast 提示和日志信息将显示为英文，而非中文
- **维护性**: 统一使用英文，降低维护复杂度，避免国际化初始化依赖问题
- **依赖关系**: 无新增依赖
