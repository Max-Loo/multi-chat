## 测试方法

### 实际加载时间测试（可选）

如需测量实际加载时间，可使用以下方法:

1. **Chrome DevTools Network Throttling**
   ```bash
   npm run dev
   # 在 Chrome DevTools 中设置 Network Throttling 为 "Fast 4G"
   # 切换语言，测量加载时间
   ```

2. **Performance API**
   ```javascript
   const start = performance.now();
   await i18next.changeLanguage('zh');
   const end = performance.now();
   console.log(`Language loading time: ${end - start}ms`);
   ```

3. **Lighthouse CI**
   ```bash
   npm run build
   npm run lighthouse
   ```
