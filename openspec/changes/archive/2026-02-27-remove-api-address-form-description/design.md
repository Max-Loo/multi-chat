## Context

当前 `ModelConfigForm.tsx` 在「API 地址」表单项下方使用 `FormDescription` 组件显示地址预览。该预览显示用户输入的值或默认的 API URL。这是对现有表单字段的简单 UI 调整。

## Goals / Non-Goals

**Goals:**
- 移除冗余的 API 地址预览，简化表单 UI
- 清理未使用的组件导入

**Non-Goals:**
- 不涉及表单验证逻辑变更
- 不涉及数据处理或 API 调用逻辑变更
- 不影响其他表单字段

## Decisions

**直接移除 FormDescription 而非替换为其他 UI 元素**

- **理由**: API 地址输入框本身已经清晰显示用户输入的内容，额外的预览文本没有提供额外价值。直接移除最简单，符合 YAGNI 原则。
- **替代方案**: 考虑过将预览改为占位符或工具提示，但这些方案会增加复杂度而没有明显收益。

## Risks / Trade-offs

**无已知风险**

这是一个简单的 UI 清理，不涉及数据流、验证或外部依赖变更。移除后不影响表单功能，只是视觉上更简洁。
