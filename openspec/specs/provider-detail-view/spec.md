# Provider Detail View

## Purpose

定义模型供应商详细信息的展开和交互行为，允许用户查看供应商的模型列表、配置元数据和进行模型搜索。

## Requirements

### Requirement: 展开供应商详细信息
系统 SHALL 支持用户展开查看单个供应商的详细信息。

#### Scenario: 展开供应商卡片
- **WHEN** 用户点击供应商卡片或"查看详情"按钮
- **THEN** 系统展开该供应商的详细信息区域
- **AND** 显示该供应商支持的所有模型列表

#### Scenario: 折叠供应商卡片
- **WHEN** 用户再次点击已展开的供应商卡片或"收起"按钮
- **THEN** 系统折叠详细信息区域
- **AND** 恢复到简洁的卡片视图

### Requirement: 显示模型列表
系统 SHALL 在详细信息视图中显示该供应商支持的所有模型。

#### Scenario: 显示模型列表
- **WHEN** 供应商详细信息被展开
- **THEN** 系统显示该供应商的所有模型
- **AND** 每个模型显示模型名称和模型 ID
- **AND** 使用列表或标签云形式展示

### Requirement: 显示供应商元数据
系统 SHALL 在详细信息视图中显示供应商的配置元数据。

#### Scenario: 显示 API 端点
- **WHEN** 供应商详细信息被展开
- **THEN** 系统显示该供应商的 API 基础 URL

#### Scenario: 显示供应商描述
- **WHEN** 供应商详细信息被展开且有描述信息
- **THEN** 系统显示该供应商的描述文本

#### Scenario: 显示官方文档链接
- **WHEN** 供应商详细信息被展开且有官方文档 URL
- **THEN** 系统显示"查看文档"链接
- **AND** 点击链接在新标签页打开官方文档

### Requirement: 支持按名称搜索模型
系统 SHALL 允许用户在展开的详细信息中搜索特定模型。

#### Scenario: 输入搜索关键词
- **WHEN** 用户在模型搜索框中输入关键词
- **THEN** 系统实时过滤模型列表
- **AND** 仅显示匹配的模型

#### Scenario: 清空搜索
- **WHEN** 用户清空搜索框
- **THEN** 系统恢复显示所有模型

### Requirement: 支持模型数量提示
系统 SHALL 在详细信息视图中显示模型数量统计。

#### Scenario: 显示模型总数
- **WHEN** 供应商详细信息被展开
- **THEN** 系统显示"共 X 个模型"的提示
- **AND** 当搜索时显示"找到 X 个模型"

### Requirement: 优化展开动画
系统 SHALL 使用平滑的动画效果提升展开/折叠体验。

#### Scenario: 展开动画
- **WHEN** 用户展开供应商详细信息
- **THEN** 系统使用 CSS transition 动画（300ms）
- **AND** 动画效果为高度展开和透明度渐变

#### Scenario: 折叠动画
- **WHEN** 用户折叠供应商详细信息
- **THEN** 系统使用 CSS transition 动画（200ms）
- **AND** 动画效果为高度收起和透明度渐变
